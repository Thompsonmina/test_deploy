About This Application

This application was developed as a solution for the Bincom Developer Test.

Technical Details:

  • Built with Python Django framework
  • Uses SQLite database (converted from provided MySQL dump)
  • Contains data for Delta State (state_id = 25)
  • Clean, responsive design with no external dependencies

Database Tables Used:

  • states - List of Nigerian states
  • lga - Local Government Areas
  • ward - Electoral wards
  • polling_unit - Individual polling units
  • party - Political parties
  • announced_pu_results - Polling unit election results
  • announced_lga_results - LGA level results (for comparison only)
